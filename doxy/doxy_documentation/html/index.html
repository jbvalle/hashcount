About The Project

This project aims to demonstrate search and sorting algorithms as well as time complexity based comparisons to determine efficiency and usability. It is implemented in two parts.

  1. Part:
  • Implementation of merge-, quick-, bubble-, and insertionsort algorithms with randomized uniquely generated numbers as inputs
  • Performance testing applying time complexity assessments to estimate performance levels
  • Evaluating performance tests based on time measurements alone.
  • Time Complexity assessment based on run time of insertionsort implementations using arrays and linked lists.
